操作说明:

1. 程序使用C++编写完成，将编译后的exe文件和itemlist.txt和subCategory.txt两个文本文件放在一个文件夹内即可执行。

2. itemlist.txt文件是商店中所有商品的列表，每行对应一种商品，每行的内容包括名称，数量单位，单价，类别和条形码（伪），数据结构如下：
    barcode,name,unit,category,subCategory,
   
    示例：
    
    ITEM000000,可口可乐,瓶,食品,碳酸饮料,3.00

    使用者可以根据需要自行添加商品，要注意每行的条形码保持唯一并且最后不能有空行。

3. subCategory.txt文件保存了所有优惠的某一子类型商品的条形码，中间不换行用","隔开，示例：
 
    ITEM000000,ITEM0000001

    使用者可以根据需要添加商品的条形码设置要按照批发价优惠的商品，注意每个条形码保持唯一最后不能有空行。

4. 对于输入相同的条形码，即分两次或多次输入了相同的商品，程序会记录该商品的总数量，输出时只显示一次。   

输入示例：
ITEM000000,ITEM000000,ITEM000003,ITEM000005

输出:

*<没钱赚商店>购物清单*

名称：可口可乐，数量：2瓶，单价：3.00(元)，小计: 6.00 (元)
名称：苹果，数量：1斤，单价：5.50(元)，小计: 5.50 (元)
名称：芒果，数量：1斤，单价：8.50(元)，小计: 8.50 (元)
____________________________________________________________

总计: 20.00 (元)
____________________________________________________________

输入示例：
ITEM000001,ITEM000001-9,ITEM000008,ITEM000006-3

输出：

*<没钱赚商店>购物清单*

名称：雪碧，数量：10瓶，单价：3.00(元)，小计: 28.50 (元)，优惠：1.50 (元)
名称：桃子，数量：3斤，单价：7.50(元)，小计: 22.50 (元)
名称：芬达，数量：1瓶，单价：3.00(元)，小计: 3.00 (元)
____________________________________________________________

批发价出售商品:
名称：雪碧，数量：10瓶
____________________________________________________________

总计: 54.00 (元) 节省: 1.50 (元)
____________________________________________________________
